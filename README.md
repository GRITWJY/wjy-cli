# wjy-cli 脚手架使用说明

本脚手架主要为自己和团队使用，每一版本功能如下

## V1.0.0
直接拉取模板(已夭折)

## V2.0.0
可选择不同的组件安装，但由于是只针对自己项目的，这个功能完善度没有vue-cli的高，一些选项也都是固定的，
但**解决了因版本号不同导致的一系列小问题，即可正常运行的版本**


## 接下来目标
补充拉取模板功能，创建后就是一个小型项目，如`登录(权限认证的框架：可选)`， `带侧边栏、导航栏案例的`， 也可以选择组件安装，
如报告批改组件、 编辑器组件等，选择之后会在侧边栏给一个选项卡
